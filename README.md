# Variables-fiscales-en-inversiones-patrimoniales
Los asesores financieros deben contemplar no solo la rentabilidad esperada de un activo, sino las implicancias fiscales que cada operación puede conllevar.
Las ganancias de capital representan una de las variables más complejas. Desde cuándo vender hasta cómo compensar, todo forma parte del análisis estratégico.
La volatilidad normativa y los debates sobre exenciones obligan a los asesores a mantener una vigilancia constante y proactiva.
